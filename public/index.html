Random German Sentence Generator

Author: Wai Man Lam

Contact: lamwaiman1988@gmail.com

Description: This website is only a proof-of-concept application with HTML/CSS/REACT on AWS with Amplify + API Gateway + Lambda + DynamoDB. It would freeze up automatically when inactive. It would start to unfreeze after 10-20 seconds of clicking the button.

It contains two parts, a frontend and a backend. The backend contains a service which provides a random German sentence. The frontend call the backend API and display a random German sentence.

Platform: This website is powered by Free Tier AWS platform
